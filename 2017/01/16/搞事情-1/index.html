数据抓取下来之后，存放在MySQL数据库。pandas提供了将数据便捷存入关系型数据库的方法，主要是以sqlalchemy方式与数据建立连接。写入数据库的主要方法是to_sql函数。但是如果直接将数据使用to_sql写进MySQL的话会报错：
MySQL Error 1170 (42000): BLOB/TEXT Column Used in Key Specification Without a Key Length
查找了一下原因：MySQL不允许在BLOB/TEXT,TINYBLOB, MEDIUMBLOB, LONGBLOB, TINYTEXT, MEDIUMTEXT, LONGTEXT,VARCHAR建索引，因为前面那些列类型都是可变长的，MySQL无法保证列的唯一性，只能在BLOB/TEXT前n个字节上建索引。
所以我们使用前20个字节作为索引就可以。于是干脆自己建表好了。
